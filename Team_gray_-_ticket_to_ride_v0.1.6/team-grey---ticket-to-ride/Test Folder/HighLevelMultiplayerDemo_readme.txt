to test the multiplayer demo set project window size to defaults

in debug menu tab (in top left) go to Customize Run Instances
enable multiple instances
set to 3

click on high_level_example scene and press run current scene in 
the top right corner or press F6

set one of the scenes to server and the other two to clients

now when on one client it moves that player for all and same 
for the other client

when done disable multiple instances and
set project window size back to 1920x1080

This was a recreation of IcyEngine's video 
"Godot Multiplayer Tutorial: The Quick and Easy High-Level API"
